**Asymptotically Unbiased Synthetic Control Methods by Moment Matching**

## Repository structure

- `mmscm.py`, core implementation of MMSCM and the benchmark methods used in the notebooks
- `dataset/`, empirical datasets used in the paper
- `result/`, cached outputs for the empirical conformal-inference notebooks
- `requirements.txt`, Python dependencies
- `LICENSE`, license file

## Setup

Create a Python environment and install the dependencies:

```bash
pip install -r requirements.txt
```

Some notebooks use `scpi-pkg`. It is already listed in `requirements.txt`.

The code-based outputs in the manuscript are Figures 2–10 and Tables 2–4. Figure 1 and Table 1 are not generated by the notebooks in this repository.

## Notebooks and paper outputs

### Main simulation figures

- `Figure2_Figure3_Simulation_Treatment_Effect.ipynb`  
  Reproduces **Figure 2** and **Figure 3**.

- `Figure4_Simulation_Distribution_MMD.ipynb`  
  Reproduces **Figure 4**.

- `Figure7_Simulation_h_G_and_Loss.ipynb`  
  Reproduces **Figure 7**. This notebook uses the same simulation framework as Figures 2 and 3 and ends with the additional comparison for the choice of `h`, the choice of `G`, and the loss function.

### Empirical figures

- `Figure5_Empirical_Counterfactual_Outcomes.ipynb`  
  Reproduces **Figure 5**.

- `Figure6_Empirical_Counterfactual_Distributions.ipynb`  
  Reproduces **Figure 6**.

### Tables and confidence-band figures

- `Tables2_3_4_Figures8_10_Compute_Intervals.ipynb`  
  Computes the treatment-effect estimates and conformal intervals used for **Tables 2–4** and **Figures 8–10**. This notebook saves intermediate NumPy files under `result/`.

- `Tables2_3_4_Figures8_10_Render_Tables_and_Confidence_Bands.ipynb`  
  Loads the saved outputs from the compute notebook, prints the LaTeX tables for **Tables 2–4**, and renders the confidence-band plots used for **Figures 8–10**.

### Supplementary notebook

- `Supplementary_Simulation_Tuning_Loss_and_SCPI_Intervals.ipynb`  
  Contains additional simulations for tuning-parameter selection, loss-function comparison, and SCPI-based interval estimation. This notebook is supplementary and is not required to reproduce the main figures and tables in the paper.

## Recommended execution order

For a full reproduction from scratch, run the notebooks in the following order:

1. `Figure2_Figure3_Simulation_Treatment_Effect.ipynb`
2. `Figure4_Simulation_Distribution_MMD.ipynb`
3. `Figure5_Empirical_Counterfactual_Outcomes.ipynb`
4. `Figure6_Empirical_Counterfactual_Distributions.ipynb`
5. `Figure7_Simulation_h_G_and_Loss.ipynb`
6. `Tables2_3_4_Figures8_10_Compute_Intervals.ipynb`
7. `Tables2_3_4_Figures8_10_Render_Tables_and_Confidence_Bands.ipynb`

The supplementary notebook can be run independently.

## Notes

- Run the notebooks from the repository root so that relative paths to `dataset/` and `result/` resolve correctly.
- The simulation notebooks can take time to finish because they loop over multiple Monte Carlo repetitions and several values of `J`.
- The empirical conformal-inference workflow is split into two notebooks so that the numerical results can be computed once and then rendered as tables and figures.
